로컬폴더를 github과 연동하기

1. 초기화 시작
	git init 

2. 계정연결
	git config --local user.name yujisu01
	git config --local user.email jssla1@naver.com

3. repository 주소 연결
	git remote add origin
		https://github.com/yujisu01/Biz_403_01_java.git

4. 로컬폴더의 프로젝트를 업로드 준비
	git add . : 현재 폴더의 모든 것을 업로드 하도록 압축하라

5. 압축된 로컬폴더를 업로드 
	git push -u origin master
